# 🏡 Housing Market Trend Analysis

This project uses Machine Learning techniques to analyze housing market trends and predict house prices based on property area. A Linear Regression model is trained using sample market data and visualized with an easy-to-understand regression graph.

## 🚀 Features

- Predict house prices using Linear Regression
- Analyze the relationship between area and price
- Visualize real market data with a regression line
- Simple and beginner-friendly implementation
